This Python script implements a **chess game** using **Pygame** and **python-chess**, managing gameplay, timers, move logging, and sound effects. Here’s a structured breakdown of the functionality:

### **1. Setup and Initialization**
- **Imports**: Includes `pygame`, `chess`, `time`, and sound-related functions.
- **Loading Resources**:
  - `load_sounds()` loads sound effects for moves, captures, and checkmates.
  - `load_images()` loads chess piece images.

### **2. User Interface Components**
- **Tkinter for saving move logs**: 
  - `ask_save_move_logs(move_log)`: Uses `filedialog.asksaveasfilename()` to save move history to a `.txt` file.

- **Pygame Window Initialization**:
  - The chess board size and UI dimensions are set using constants.
  - The window (`win`) is created using `pygame.display.set_mode((WIDTH, HEIGHT))`.

### **3. Main Game Function: `main(selected_opponent)`**
- **Chess Board Setup**: Uses `chess.Board()`, handling legal moves and tracking pieces.
- **Timers**: Initializes **5-minute countdowns** for white and black.
- **Game Message Handling**: Displays status messages like **“White to move”** or **checkmate notifications**.

### **4. Event Handling (Mouse Clicks)**
- **Click on Top Bar Buttons**:
  - `Restart`: Restarts the game.
  - `End`: Saves move logs and exits.
  - `Mute`: Toggles sound effects.

- **Drag-and-Drop Chess Piece Moves**:
  - Detects **mouse down** to select a piece.
  - Detects **mouse up** to attempt a move.
  - Plays appropriate sounds (move, capture, promotion, check).
  - Updates the `move_log` for game history.

### **5. Game Ending Conditions**
- Checks if:
  - **Checkmate**: Declares a winner.
  - **Stalemate**: Draw due to no legal moves.
  - **Insufficient Material**: Draw if not enough pieces left.
  - **75-Move Rule**: Draw if neither player captures or moves a pawn within 75 moves.
  - **Fivefold Repetition**: Draw if the same board position occurs five times.

### **6. Drawing the Chess Board**
- **Game board rendering** using:
  - `draw_game_board()`
  - `highlight_squares(board, selected_square)`
  - `draw_pieces(board, drag_info)`
  - `draw_sidebars(move_log, white_time, black_time)`
  - `draw_bottombar(message)`

### **7. Running the Game**
- The `run_game(win)` function checks if the game should start from the **welcome screen** or directly into **gameplay**.

### **Conclusion**
This script provides a fully functional **chess game** with:
✔ UI interactions  
✔ Move logging  
✔ Sounds  
✔ Chess logic (checkmate, stalemate, promotions, legal moves)
