This Python script manages the **opponent selection screen** for a chess game using **Pygame**. Here's a structured breakdown of its functionality:

---

### **1. Setup and Initialization**
- **Imports**:
  - `pygame` for UI rendering and event handling.
  - `WIDTH` and `HEIGHT` from `config` for screen dimensions.
  - `sound` module for playing button click sounds.

- **Load Sounds**: `sounds = sound.load_sounds()` ensures sound effects are available.

---

### **2. Colors and Button Class**
- **Color Definitions**:
  - `WHITE`, `BLACK`, `GRAY`, `DARK_GRAY` are used for UI elements.

- **Reusable Button Class** (`Button`):
  - Stores button properties (rectangle, text, colors, font).
  - `draw(surface)`: Draws the button with a border.
  - `is_clicked(pos)`: Checks if a mouse click occurs inside the button's rectangle.

---

### **3. Rendering the Opponent Selection Screen**
**`draw_choose_opponent(win)`**:
- Loads **background image** (`bg_2.jpg`).
- Displays a **title**: “Choose an opponent to play against”.
- Creates **buttons**:
  - **"Human"** (Play against another player).
  - **"Computer"** (Play against AI).
  - **"Back"** (Return to the welcome screen).

---

### **4. Handling User Input**
**`handle_choice_events(event)`**:
- Creates **rectangles** for buttons, matching `draw_choose_opponent()`.
- Listens for:
  - `pygame.QUIT`: Exit game.
  - `MOUSEBUTTONDOWN`: Detects button clicks, triggers appropriate action:
    - `"game", "human"` → Start a **human vs. human** chess game.
    - `"game", "computer"` → Start a **player vs. AI** chess game.
    - `"welcome", None"` → Go **back** to the welcome screen.
- Plays **click sounds** for button interactions.

---

### **Summary**
✔ Displays a **selection screen** for choosing an opponent  
✔ Uses a **reusable button class** for flexibility  
✔ Handles **click events** to transition between game states  
✔ Plays **sound effects** for an interactive experience
