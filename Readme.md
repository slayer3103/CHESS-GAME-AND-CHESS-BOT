# ♟️ Chess Game & AI Chess Bot

## 📌 Overview

A fully functional **Chess Game with an AI-powered Chess Bot**, designed to simulate real-world gameplay with intelligent decision-making. This project demonstrates core concepts of **game development, algorithms, and artificial intelligence**, making it a strong showcase of problem-solving and system design skills.

---

## 🚀 Features

- Interactive chessboard with complete rule implementation
- AI-powered opponent using decision-making algorithms
- Move validation including check, checkmate, and stalemate detection
- Smooth user interface for seamless gameplay experience
- Modular and scalable code structure

---

## 🛠️ Tech Stack

- **Programming Language:** Python
- **Libraries/Frameworks:** Pygame
- **Concepts Used:**
  - Game Development
  - Artificial Intelligence (Minimax Algorithm, Evaluation Functions)
  - Data Structures & Algorithms

---

## 🧠 How It Works

The chess bot is powered by the **Minimax Algorithm**, which evaluates possible moves and predicts future outcomes to select the optimal move.

- Explores possible game states recursively
- Uses evaluation functions to score board positions
- Applies optimization techniques (like pruning) for efficiency

## ▶️ Getting Started

### Prerequisites

- Python 3.x installed

### Installation

```bash
git clone https://github.com/your-username/chess-game.git
cd chess-game
pip install -r requirements.txt
```

### Run the Game

```bash
python main.py
```
